# Mastra AI Excel VBA Generator - Windows版

このアプリケーションは、チャットとExcelファイルをもとに、複数のLLM（GPT-4 Turbo、Gemini 2.0、Claude 3、Mixtral）を連携させ、自動的にExcelファイルへの入力を行うVBAコードを生成するWindows用アプリケーションです。

## システム要件

- Windows 10/11
- Python 3.8以上（必須）
- インターネット接続

## インストール方法

1. このZIPファイルをWindows PCの任意の場所に解凍します。
   - **重要**: 必ずすべてのファイルを同じフォルダに解凍してください。
   - フォルダ構造を維持したまま解凍してください（右クリック→「すべて展開」を選択）。

2. Pythonがインストールされていない場合は、[Python公式サイト](https://www.python.org/downloads/)からダウンロードしてインストールしてください。
   - インストール時に「**Add Python to PATH**」にチェックを入れることを忘れないでください。
   - インストール完了後、コンピュータを再起動することをお勧めします。

3. `start_windows.bat` ファイルを**右クリックして「管理者として実行」**を選択します。
   - 初回実行時は必要なパッケージがインストールされます。
   - インストールには数分かかる場合があります。

## 使用方法

1. `start_windows.bat` を実行すると、自動的にブラウザが開きアプリケーションが起動します。
   - バックエンドサーバーの起動には30秒程度かかる場合があります。
   - 「サーバーの起動を待っています...」というメッセージが表示されている間は、しばらくお待ちください。

2. チャットでプロジェクト情報を入力するか、Excelファイル(.xlsm)をアップロードします。
3. 確定事項（工事名、施工場所、工期、作業者数）を入力フォームに入力します。
4. システムが自動的に3回以上の質問を行い、必要な情報を収集します。
5. 生成されたVBAコードをコピーしてExcelに貼り付けます。

## 機能

### UI機能

- チャット送信可能なUI
- Excelファイル(.xlsm)のアップロード機能
- 確定事項入力フォーム（工事名、施工場所、工期、作業者数）

### LLM連携機能

以下のLLMを使用し、それぞれの役割を設定：

- GPT-4 Turbo：チャット内容の論理的構造化
- Gemini 2.0：Excelデータ構造の分析
- Claude 3：表現の曖昧さを明確化
- Mixtral：細かいニュアンスや例外処理の補完

### VBAコード生成

生成された最終情報をもとに、準備作業→本作業→後始末作業の順に、Excelファイルに入力するためのVBAコードを作成します。

## トラブルシューティング

- **「このアプリはお使いのPCでは実行できません」というエラーが表示される場合**：
  - このパッケージはPythonベースのアプリケーションであり、exeファイルではありません。
  - `start_windows.bat`を**右クリックして「管理者として実行」**を選択してください。

- **「localhost:1337が見つかりません」というエラーが表示される場合**：
  - バックエンドサーバーが正常に起動していない可能性があります。
  - コマンドプロンプトのエラーメッセージを確認してください。
  - 以下の手順を試してください：
    1. アプリケーションを完全に終了する（コマンドプロンプトウィンドウを閉じる）
    2. タスクマネージャーを開き、「Python」または「python.exe」のプロセスが残っていれば終了する
    3. ブラウザを一度閉じる
    4. `start_windows.bat` を右クリックして「管理者として実行」を選択
    5. サーバーの起動が完了するまで（60秒程度）待つ
    6. 自動的にブラウザが開かない場合は、手動でブラウザを開き http://localhost:1337 にアクセス
  - ポート1337が他のアプリケーションで使用されていないか確認してください：
    1. コマンドプロンプトを管理者として実行
    2. `netstat -ano | findstr :1337` を実行
    3. 結果が表示される場合、そのポートは使用中です。アプリケーションを終了するか、別のポートを使用してください。

- **アプリケーションが起動しない場合**：
  - Pythonがインストールされていることを確認してください。
  - コマンドプロンプトで`python --version`を実行して、Pythonがインストールされていることを確認してください。
  - Pythonのインストール時に「Add Python to PATH」にチェックを入れたことを確認してください。
  - コンピュータを再起動してから再度試してみてください。

- **ブラウザが自動的に開かない場合**：
  - バックエンドサーバーが起動するまで30秒程度待ってから、手動で http://localhost:1337 にアクセスしてください。

- **バックエンドサーバーが起動しない場合**：
  - コマンドプロンプトでエラーメッセージを確認してください。
  - 必要なパッケージがインストールされていない可能性があります。手動でインストールするには：
    ```
    cd backend
    pip install -r requirements.txt
    ```
  - ファイル構造が正しいか確認してください。フォルダ構造は以下のようになっているはずです：
    ```
    mastra_ai_portable_windows/
    ├── backend/
    │   ├── app/
    │   │   ├── main.py
    │   │   └── ...
    │   └── requirements.txt
    ├── launcher.py
    ├── start_windows.bat
    └── README_windows.md
    ```

## 注意事項

- OpenRouter APIキーは既に組み込まれているため、追加設定は不要です。
- アプリケーションの実行中はインターネット接続が必要です。
- このアプリケーションはPythonを使用して実行されます。
- アプリケーションの起動には管理者権限が必要な場合があります。
