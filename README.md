# Mastra AI Excel VBA Generator

複数のLLM（GPT-4 Turbo、Gemini 2.0、Claude 3、Mixtral）を連携させ、自動的にExcelファイルへの入力を行うVBAコードを生成するWindows11用アプリケーション。

## LLM連携機能

このアプリケーションはOpenRouter APIを通じて以下の4つのLLMモデルを連携させています：

1. **GPT-4 Turbo**: チャット内容の論理的構造化
   - ユーザーとの会話から重要な情報を抽出・整理
   - 最終的なVBAコード生成にも使用

2. **Gemini 2.0**: Excelデータ構造の分析
   - アップロードされたExcelファイルのデータ構造を詳細に分析
   - セル範囲の内容を把握し最適なVBAコードのための情報を提供

3. **Claude 3**: 表現の曖昧さを明確化
   - ユーザー入力や抽出されたデータの曖昧な表現を特定し明確化
   - より正確なVBAコード生成をサポート

4. **Mixtral**: 細かいニュアンスや例外処理の補完
   - 特殊なケースや例外的な状況を特定し、対処方法を提案
   - VBAコードの堅牢性向上に貢献

## 主な機能

1. **インタラクティブなチャットUI**: React製のUIでユーザーとの対話を実現
2. **Excelファイル解析**: 指定されたセル範囲を自動解析してナレッジベースを構築
3. **3段階の質問プロセス**: 基本情報→詳細情報→最終確認の流れで必要な情報を収集
4. **VBAコード生成**: 収集した情報をもとに準備作業→本作業→後始末作業の順にコードを生成
5. **スタンドアロンアプリケーション**: Windows 11で動作する単体アプリケーション
