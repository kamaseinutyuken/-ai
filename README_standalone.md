# Mastra AI Excel VBA Generator - スタンドアロンアプリケーション

このアプリケーションはMastra AIを使用して複数のLLM（GPT-4 Turbo、Gemini 2.0、Claude 3、Mixtral）を連携させ、
Excelファイルの自動入力とVBAコード生成を行うWindows用スタンドアロンアプリケーションです。

## 特徴

- **サーバー不要**: インターネット接続のみでローカルで完全に動作
- **簡単操作**: インストール不要で、exeファイルをダブルクリックするだけで起動
- **高度なAI機能**: 複数のLLMを連携させた高度な処理
- **Excelファイル処理**: Excelファイルの解析と編集
- **VBAコード生成**: 自動VBAコード生成機能

## インストール方法

1. `mastra_ai_standalone.zip`ファイルをダウンロード
2. お好きな場所に解凍
3. 解凍したフォルダ内の`Mastra AI Excel VBA Generator.exe`をダブルクリック

## 使用方法

1. アプリケーションを起動
2. チャットインターフェースでプロジェクト情報を入力、またはExcelファイルをアップロード
3. アプリケーションが自動的にデータを分析
4. 生成されたVBAコードを取得し、Excelに適用

## 必要環境

- Windows 10/11
- インターネット接続 (OpenRouter APIへのアクセス用)

## トラブルシューティング

- **アプリケーションが起動しない場合**:
  - Windowsのセキュリティ警告が表示された場合は「詳細情報」→「実行」をクリック
  - アンチウイルスソフトが干渉している場合は、例外に追加
  
- **APIエラーが発生する場合**:
  - インターネット接続を確認
  - OpenRouter APIキーが有効か確認（設定で変更可能）

## 技術仕様

このアプリケーションは以下の技術を使用しています：
- Python
- PyQt6 (GUIフレームワーク)
- FastAPI (内部API処理)
- React (UI)
- OpenRouter API (LLM接続)
